# Game Tracker v0.1

Got tired of the game tracking tools available so decided to make my own that's simple and straightforward. Will add more features as needed. Selections are stored locally so selections will only persist as long as you use the same browser and don't clear your cache (I think).

This is a static app that is NOT pulling from any sources dynamically so feel free to clone but you'll want to get an AI tool to revise the list based on your preferences. Ask it something like "replace all the game entries with the revelant content for games that are [from a certain genre / top 25 from each year for the past 5 years / top 50 indie games from the past 10 years]". It should do it without issue. Whatever works for you.

Each game's row has a button that takes you to relevant videos on YouTube, let's you add games to a shortlist, mark games as played, shows you how long it will take to beat on average pulled from HowLongToBeat, MetaCritic score, and a short description and genre.
